Sonu Kirana & General Store 🛒
A complete, professional, and responsive grocery store website built for Sonu Kirana & General Store. The project features a public-facing customer portal and a secure admin dashboard for inventory management.

🌟 Features
Customer Portal
Modern UI/UX: Premium green-themed design, smooth animations, and a responsive layout that looks great on all devices (mobile, tablet, desktop).
Dynamic Hero Slider: Eye-catching image carousel with quick action buttons.
Customer Testimonials: Mobile-friendly, swipeable horizontal scroll layout displaying authentic customer reviews.
Products & Offers: Real-time display of grocery products and special offers fetched directly from the database.
Contact Form: An integrated form and location map for easy customer communication.
Admin Dashboard (Restricted Access)
Secure Authentication: Firebase-powered login strictly for the store owner.
Inventory Management: Easy-to-use CRUD (Create, Read, Update, Delete) interface to manage store products.
Live Updates: Changes made in the admin panel instantly reflect on the customer website.
🛠️ Technologies Used
Frontend: HTML5, Vanilla CSS, JavaScript (ES6+).
Framework: Bootstrap 5 (for responsive grids, navbar, and carousels).
Backend & Database: Firebase Authentication and Cloud Firestore.
Icons & Avatars: Bootstrap Icons and dynamically generated UI Avatars for testimonials.
